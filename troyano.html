Concepto

Un troyano es un malware que se disfraza como software legítimo para engañar al usuario y obtener acceso al sistema.

Dónde se puede contraer

  • Descargas de software pirata
  • Aplicaciones falsas
  • Archivos adjuntos maliciosos
  • Sitios web fraudulentos

Cómo detectar el ataque

  • El equipo se ralentiza sin razón aparente
  • Programas que se inician automáticamente
  • Cambios en configuración del sistema
  • Alto consumo de recursos
  • Conexiones de red no autorizadas

Cómo combatirlo

  • Instalar antivirus confiable
  • Mantener firewall activado
  • Solo descargar software de fuentes oficiales
  • Verificar permisos de aplicaciones
  • Escanear regularmente el sistema

Qué hacer si fuiste víctima

  • Desconectar de internet
  • Ejecutar análisis antivirus completo
  • Eliminar programas sospechosos
  • Cambiar todas las contraseñas
  • Monitorear actividad bancaria por posibles fraudes

Recomendaciones generales de prevención

  • Usar autenticación de dos factores
  • Mantener software actualizado
  • No usar software pirata
  • Verificar remitentes de correos
  • Realizar copias de seguridad periódicas
  • Educar a usuarios sobre seguridad
  • Usar contraseñas fuertes y únicas
