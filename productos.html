TIPOS DE CIBERATAQUES

ADWARE

Es un software que muestra anuncios no deseados en tu dispositivo, generalmente en forma de ventanas emergentes o que redireccionan a sitios web de publicidad.

Iniciar Ahora

PISHING

es un tipo de ciberataque donde los atacantes se hacen pasar por entidades confiables para robar información sensible, como contraseñas y datos bancarios.

Iniciar Ahora

RANSOMWARE

El ransomware es un tipo de malware que cifra los archivos del usuario y exige un rescate (usualmente en criptomonedas) para recuperar el acceso a los datos.

Iniciar Ahora

TROYANO

Un troyano es un malware que se disfraza como software legítimo para engañar al usuario y obtener acceso al sistema. Iniciar Ahora
